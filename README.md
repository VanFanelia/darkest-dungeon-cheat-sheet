# Darkest Dungeon Cheat Sheet

A comprehensive, printable cheat sheet for **Darkest Dungeon** featuring curios, provisions, and dungeon-specific information.

## What is this?

This repository contains a single-page HTML cheat sheet designed to help players quickly reference important game information while playing Darkest Dungeon. The cheat sheet includes:

- **Curios for all dungeons** (Ruins, Weald, Warrens, Cove)
- **Item interactions** with each curio
- **Provisions recommendations** per dungeon type
- **Optimized for printing** in A4 landscape format

## Usage

Simply open `index.html` in your web browser. The page is optimized for printing and can be used as a quick reference guide during gameplay.

## Files

- `index.html` - The main cheat sheet (HTML format)
- `darkest-dungeon-cheat-sheet.pdf` - Pre-rendered PDF version
- `image/` - Local copies of all curio images
